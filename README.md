# Bit-cora-Proyecto-Grupal-1-Prestamos-de-Libros-
Análisis de una matriz de datos de una librería en la gestión de prestamos de libros

# Bitácora del Proyecto

  1.-Conocimiento y Calidad de los datos extraidos: Fuentes directas a bases de datos, o son producto de una extracción inicial sujetas o ono a un filtrado, etc., fecha de la data para el analisis requerido
  2.-Entendimiento inical de los datos:
    -Definir cual es el objetivo del Analisis Ejemplo:  ¿Que preguntas quieres responder?, ¿Plantemineto de una Hipotesis o afirmación que quieras confirmar?.
  3.-Identificar las variables / grupos de datos o Categorias
    -Valores Numericos, texto, alfanumericos
  4.-Creacion de tablas dinamicas que permitan cruces de la informacón y proporcionen mayor entendimiento de los datos e información analizada
  5.-Elaboración de graficos que permitan a simple vista un respuestas al analisis definido
  6.-Informe o Presentacion de los analisis para posterior discusión y divulgacion de las conclusiones o sugerencias
