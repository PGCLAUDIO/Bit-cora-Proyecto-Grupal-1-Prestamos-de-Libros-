# Bit-cora-Proyecto-Grupal-1-Prestamos-de-Libros-
Análisis de una matriz de datos de una librería en la gestión de prestamos de libros

# Bitácora del Proyecto

  - Conocimiento y Calidad de los datos extraidos: Fuentes directas a bases de datos, o son producto de una extracción inicial sujetas o ono a un filtrado, etc., fecha de la data para el analisis requerido
  - Entendimiento inical de los datos:
    - Definir cual es el objetivo del Analisis Ejemplo:  ¿Que preguntas quieres responder?, ¿Plantemineto de una Hipotesis o afirmación que quieras confirmar?.
    - Identificar las variables / grupos de datos o Categorias: Valores númericos, texto, alfanúmericos.
    - Calidad de los datos: errores, espacios vacios, si todos los elementos de extracción estan completos, etc.
  - Creación de tablas dinamicas que permitan cruces de la informacón y proporcionen mayor entendimiento de los datos e información analizada
  - Elaboración de graficos que permitan a simple vista un respuestas al analisis definido
  - Informe o Presentacion de los analisis para posterior discusión y divulgación de las conclusiones o sugerencias
